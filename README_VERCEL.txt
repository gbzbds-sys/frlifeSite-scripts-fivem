SITE_3D_FRLIFE - Déploiement Vercel
===================================

Ce dossier est prévu pour ton site Vercel.

À envoyer sur GitHub :
- index.html
- assets/
- README.txt
- README_VERCEL.txt
- README_CORRECTION.txt

Ne pas envoyer sur GitHub/Vercel :
- les ZIP des scripts
- les ressources FiveM escrow
- les fichiers server.lua/client.lua des scripts

Les scripts doivent être vendus/livrés via Tebex avec :
- FiveM Asset = script CFX Keymaster
- File Download = documentation client uniquement

Après upload sur GitHub, Vercel relance automatiquement le déploiement.
Attends 1 à 3 minutes, puis fais CTRL + F5 sur le site.
