FRLife Tracker V3.3.1

Système RP complet pour poser un tracker GPS sur un véhicule, gérer les traceurs avec un boîtier de contrôle et utiliser un anti-tracker avec scan mécanique, timer visible et brouillage temporaire du signal.

Framework : Qbox / QBCore
Inventaire : ox_inventory
Prix : 24,99€
Version : 3.3.1 Escrow

Fonctionnalités principales

Le tracker sert à poser un traceur sur un véhicule. Le boîtier sert uniquement à gérer les trackers posés : localiser, faire sonner, déconnecter et supprimer les trackers déconnectés.
Tracker GPS
Pose discrète sur véhicule via item, avec menu NUI 3D bleu, nom du traceur, type de véhicule, chrono et mode discret.
Boîtier de contrôle
Gestion des traceurs actifs avec localisation, bip sonore, déconnexion et suppression des anciens trackers.
Anti-tracker
Scan mécanique RP avec interface NUI 3D rouge, détection de traceur et résultat visible.
Brouillage temporaire
Le signal peut être brouillé de 1 seconde à 1 minute. Pendant ce temps, le poseur ne peut plus localiser ni faire sonner le tracker.

Images et items Tracker

Les images sont incluses dans le ZIP Tebex. Elles sont à copier dans ox_inventory/web/images. La configuration item est fournie dans la documentation client. Des captures sont aussi affichées pour la pose du tracker, le boîtier de gestion et l’anti-tracker.

    label = "Tracker GPS",
    weight = 1500,
    stack = true,
    close = true,

        event = "frlifetracker:client:useTracker"

    weight = 500,

    description = "Boîtier permettant de gérer les trackers GPS actifs.",
    client = {
        event = "frlifetracker:client:useTrackerPanel"

},

["antitrakeur"] = {
    label = "Anti-tracker",
    weight = 500,
    stack = true,
    close = true,
    description = "Détecteur permettant de scanner et brouiller un tracker.",
    client = {
        event = "frlifetracker:client:useAntiTracker"
    }
},
Installation Tracker
1. Acheter le script sur Tebex.
2. Récupérer l’accès FiveM Asset via Cfx/Keymaster.
3. Télécharger le ZIP client Tebex avec README, images et configuration items.
4. Mettre la resource FRLifeTracker dans resources/[frlife]/.
5. Copier les images dans ox_inventory/web/images/.
6. Ajouter la configuration items dans ox_inventory/data/items.lua.
7. Ajouter les ensures nécessaires dans server.cfg selon la documentation fournie.
8. Redémarrer le serveur et tester sur un serveur de développement.

FRLife Bombe RP

Système RP de bombe véhicule avec pose via item, boîtier déclencheur, panel NUI 3D rouge, localisation, bip, déconnexion, suppression et détonation avec chrono configurable.

Framework : Qbox / QBCore
Inventaire : ox_inventory
Prix : 19,99€
Livraison : FiveM Asset + ZIP installation

Fonctionnalités Bombe

Pose de bombe
Menu NUI 3D rouge pour configurer la pose sur véhicule : nom RP, type de véhicule, référence, mode discret et chrono.
Boîtier déclencheur
Le boîtier permet de gérer les bombes déjà posées : localiser, faire biper, déconnecter, supprimer et déclencher.
Détonation avec timer
Déclenchement immédiat ou programmé avec un timing configurable de 1 à 60 secondes et chronomètre visible dans le panel.
Gestion propre
Les bombes déconnectées peuvent être supprimées de la liste pour garder un panel propre.

📻 FRLife Radio - Script Radio FiveM

Script radio FiveM conçu pour les serveurs RP. Le joueur utilise l’item Radio depuis ox_inventory pour ouvrir l’interface radio et rejoindre une fréquence. Livraison propre via Cfx/Keymaster avec un ZIP Tebex contenant l’image, l’item et la documentation client.

Framework : Qbox / QBCore selon configuration
Inventaire : ox_inventory
Utilisation : item Radio
Livraison : FiveM Asset + ZIP installation

Image et item Radio

L’image radio.png est incluse dans le ZIP Tebex. Elle est à copier dans ox_inventory/web/images. Le code item est fourni pour ox_inventory/data/items.lua.
Radioradio.png

Item radio utilisable depuis l’inventaire pour ouvrir le menu radio.

Code ox_inventory/data/items.lua — Radio
["radio"] = {
    label = "Radio",
    weight = 300,
    stack = false,
    close = true,
    description = "Permet d’ouvrir la radio et de rejoindre une fréquence.",
    client = {
        image = "radio.png",
        event = "frlife_radio:useRadio"
    }
},
Installation Radio
1. Acheter le script sur Tebex.
2. Récupérer l’accès FiveM Asset via Cfx/Keymaster.
3. Télécharger le ZIP Tebex avec README, image radio.png et item ox_inventory.
4. Télécharger la resource frlife_radio depuis les assets Cfx/Keymaster.
5. Mettre la resource frlife_radio dans resources/[frlife]/.
6. Copier radio.png dans ox_inventory/web/images/.
7. Copier le code item dans ox_inventory/data/items.lua.
8. Ajouter dans server.cfg :

ensure ox_lib
ensure ox_inventory
ensure frlife_radio

9. Redémarrer le serveur.
10. Tester avec /giveitem ID radio 1 puis utiliser l’item Radio dans l’inventaire.
